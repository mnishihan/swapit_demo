SwapIt Demonstration
====================

This module is a simple demonstration of how to build a custom module utilizing
the nodeJS integration module.

Setup
=====

1.  Install and enable the nodejs integration module
    http://drupal.org/project/nodejs along with the appropriate node server
    configuration.

2.  Enable the swapit demonstration module.

3.  Make sure you're node server is running.

4.  Go to mysite.com/swap and check it out!  Use a couple of sessions the
    node-ey goodness.
